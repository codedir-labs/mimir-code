# Themes

Configure your UI theme in `.mimir/config.yml`:

```yaml filename=".mimir/config.yml"
ui:
  theme: mimir  # See available themes below
```

Or use `/theme <name>` to switch at runtime.

## Custom Themes

Create YAML files in `~/.mimir/themes/` (global) or `.mimir/themes/` (project-specific).

### Required Structure

```yaml filename="~/.mimir/themes/my-theme.yml"
name: My Theme
supportsFullColor: true  # false for ANSI-only terminals

colors:
  # 47 required properties - see built-in themes for reference
  headerBg: { bg: '#1a1a1a', fg: '#ffffff' }
  headerText: { fg: '#e0e0e0' }
  # ... (all properties must be defined)
```

### Color Properties

- `fg` - Foreground color (hex: `#RRGGBB`)
- `bg` - Background color (hex: `#RRGGBB`)
- `bold`, `dim`, `italic`, `underline` - Text modifiers (optional)

View built-in theme files in `src/shared/config/themes/definitions/` for complete examples.
