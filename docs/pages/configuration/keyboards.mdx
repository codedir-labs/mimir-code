# Keyboard Shortcuts

<Callout type="warning">
  **TODO**: This page is under construction. Keyboard shortcuts documentation will be added soon.

Configuration for keyboard shortcuts can be found in `.mimir/config.yml` under the `keyBindings` section.

```yaml filename=".mimir/config.yml"
keyBindings:
  interrupt: [Ctrl+C, Escape]
  accept: Enter
  showTooltip: [Ctrl+Space, Tab]
  # ... more keybindings
```
